# ascii-cinema
Simple Java program implemented as a lecture for the highshcool I teach in. 
It allows to manage a 2D cinema, booking seats and canceling, saving the state of the theatre into a txt file.
It has an ascii interface to represent the theatre:

Inserire prossima operazione:
	- effettua prenotazione (a)
	- cancella prenotazione (s)
	- salva ed esci (q)

The aim of the lecture is to
- understand how to manage 2D arrays
- understand the importance of global constants
- learn basics of method definition and usage
- get familiar with try-catch statement and possibly learn how to create personalized exceptions and throw them within methods
- get familiar with streams and files.

The program is developed in a structured programming paradigm, without use of dynamic arrays.
