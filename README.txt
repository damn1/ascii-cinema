# ascii-cinema
Simple Java program to manage a 2D cinema, booking seats and canceling, saving the state of the theatre into a txt file.

Inserire prossima operazione:
	- effettua prenotazione (a)
	- cancella prenotazione (s)
	- salva ed esci (q)

	fila (1 - 6) ?: 1
	posto (1 - 10): 3
